Arrow functions

Las Arrow functions o funciones flecha son una forma corta de escribir funciones que aparece en Javascript a partir de ECMAScript 6.

Hacen que el código sea mucho más legible y claro de escribir, mejorando la productividad y la claridad a la hora de escribir código.

Algunas ventajas a la hora de simplificar código son:

  • Si el cuerpo de la función sólo tiene una línea, podemos omitir las llaves.
  • La fecha automáticamente hace un return, por lo que podemos omitir la palabra return.
  • Si la función tiene solo un solo parámetro, se pueden omitir los paréntesis
