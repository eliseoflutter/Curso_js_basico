Arreglos

Los arreglos son objetos y se parecen a una lista, las cuales nos permiten guardar varias variables y acceder a ellas de manera independiente, es como tener una variable con distintos compartimentos donde podemos introducir datos distintos. Para ello utilizamos un índice que nos permite especificar el compartimiento o posición a la que nos estamos refiriendo. Los arrays en Javascript empiezan siempre en la posición 0.
